function ImportCSV(XLS: TXLSReadWriteII5; SheetIndex,Col,Row: integer; Filename: AxUCString; SepChar: AxUCChar; HasQuoteChar: boolean): boolean; overload;

This routine will import a text (CSV) file into a worksheet. Any existing cells on the worksheet will be overwritten. Correct value for decimal separator is detected automatically.
