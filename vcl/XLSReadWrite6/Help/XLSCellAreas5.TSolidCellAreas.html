Declaration

type TSolidCellAreas = class(TBaseCellAreas)

Description

Base class for objects that manipulates cell areas. Solid areas are areas that not can be splitted by delete/insert/move operations, such as merged cells.

procedure Copy(const Col1,Row1,Col2,Row2: integer; const DeltaCol,DeltaRow: integer); override;

Copies cell areas. Copies the cell areas that intersects the area given by Col1,Row1, Col2,Row2. The areas are copied DeltaCol and DeltaRow offset from their current position. True if a Copy of Move operation failed because one of the areas was to be copied over an existing area. Equivalen to the Excel error message "Can not change part of a merged cell".
