property DirectRead: boolean read GetDirectRead write SetDirectRead;

When DirectRead mode is active, the cell values are not stored in memory. Instead is the OnReadCell event fired when a cell is read. You can then access the cell value in the Xc12Manager5.TXLSEventCell object, ACell parameter, in the event.
The advantage with the DirectRead mode is that no memory is used for storing cell values. As Excel 2007+ files can have 100 of millions of cells this can be a greate memory save. If you know that you just want a few cells, or going to store them elsewhere, this is a good choice. See also DirectWrite.

property DirectWrite: boolean read GetDirectWrite write SetDirectWrite;

DirectWrite mode writes cell values directly to the disk file instead of first storing them in memory. This can save hughe ammount of memory if you are writing large files. See also DirectRead.
