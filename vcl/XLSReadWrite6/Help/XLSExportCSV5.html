procedure ExportCSV(XLS: TXLSReadWriteII5; SheetIndex: integer; SeparatorChar, QuoteChar: WideChar; Options: TExportCSVOptions; Filename: AxUCString);

This routin will export the content of a worksheet to a text (CSV) file. ˜param XLS The XLSReadWriteII object to export. ˜param SheetIndex Index of the work sheet to export. ˜param QuoteChar Quote character for text strings. ˜param SeparatorChar The character used to separate the cell values (fields) in the file. Common used characters are comma, semicolon and tab. ˜param Options File export options. ˜param Filename The name of the file that the cells shall be exported to.

Types

TExportCSVOption = (...);

Options when exporting to CSV files.

Values
  • ecoQuotedStrings:  
  • ecoFormatValues: Put the content of string cells withing quotes.
  • ecoSelection: Apply number formatting to number cells. When not used, a number value is converted to a string with the FloatToStr function.
  • ecoFormulas: Export only the current selection. See ˜[link SheetData4.TSheet.SelectedAreas TSheet.SelectedAreas]. If there is more than one selected area, all of the will be written to the file.
TExportCSVOptions = set of TExportCSVOption;

Write the formulas to the file, instead of the values (result) of the formulas.
